# portfolio

個人事業者・小規模チーム向け業務Webアプリ制作の営業用ポートフォリオサイト(単一ページ)。

## 構成

- `index.html` — 単一ページ(CSS/JSインライン、スマホファースト)
- `img/` — 実績スクリーンショット(ダミーデータ)

## ローカル確認

```bash
python3 -m http.server 8000
# http://localhost:8000/
```

## デプロイ

`main` への push で Vercel が自動デプロイします。
本番 URL: https://portfolio-delta-five-0xwqrs77eh.vercel.app
