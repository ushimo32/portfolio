# スクリーンショット

実績セクションで使う画面キャプチャ。index.html の `.shots` 内 `figure.shot` から参照している。

| 順 | ファイル | キャプション |
| --- | --- | --- |
| 01 | `01-dashboard.png` | 毎朝の状況をひと目で把握するダッシュボード |
| 02 | `02-cattle-list.png` | 頭数・治療状況をスマホで一覧管理 |
| 03 | `03-morning-sheet.png` | 朝礼シート:昨日の変化・今日やること・シフトを1枚に自動集約 |
| 04 | `04-analysis.png` | 経営分析:利益率や出荷成績を自動集計 |

※ 牧場名・個人名・実データが写り込まないダミーデータの画面であること。差し替える際は同じファイル名で上書きすれば index.html の変更は不要。
